# 蓝图类中传递

使用call就可以直接调用

# 跨蓝图类传递函数

有的蓝图A有fa方法，有的蓝图B有fb方法，如何在B蓝图中调用A蓝图的方法

## 控制台CE命令

CE +  场景的函数 可以在控制台执行函数进行作弊

- 只能调用场景中的事件
- 只能在viewport 模式下调试

蓝图类中的EXECUTE CONSOLE COMMAND方法可以执行控制台命令

## 控制台KE命令

CE +  类型名 + 场景的函数 向同样类型当中广播这个事件

- 只能调用蓝图类中的事件
- 只能在viewport 模式下调试

可以用这个方法输入秘籍，但是执行效率很低

## 事件分发器/发报机

绑定就是注册事件，

执行发报机，会执行上面注册的所有事件，而且不用关心接受事件的到底是什么，按照注册的顺序执行

原则上在哪注册的，也需要在对应的地方取消。

可以调用传入对象公开的方法，也可以传别的对象
